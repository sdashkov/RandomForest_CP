# Курсовая работа  
## Разработка программных средств поиска и анализа киберугроз  
7 семестр

Данный репозиторий содержит курсовую работу по дисциплине  
**«Разработка программных средств поиска и анализа киберугроз»**,  
посвящённую применению методов машинного обучения для обнаружения и классификации сетевых атак.

---

## Описание проекта

В рамках данной работы задача обнаружения киберугроз рассматривается как задача **многоклассовой классификации сетевого трафика**.  
Для её решения применяется алгоритм **Random Forest (Случайный лес)**, позволяющий эффективно работать с большим числом признаков и обеспечивающий высокое качество классификации.

---

## Цель и задачи работы

### Цель
Разработка и обучение модели машинного обучения для выявления и классификации киберугроз на основе характеристик сетевого трафика.

### Основные задачи
- Анализ предметной области и типов сетевых атак  
- Предобработка и анализ исходных данных  
- Обучение модели машинного обучения  
- Подбор оптимальных гиперпараметров  
- Оценка качества модели с использованием метрик многоклассовой классификации  
- Демонстрация применения модели для обнаружения киберугроз  

---

## Используемые методы и алгоритмы

- Случайный лес (Random Forest)
- Деревья решений
- Бэггинг и бутстрэп-выборки
- Анализ признаков и корреляционный анализ
- Подбор гиперпараметров (GridSearch / RandomizedSearch)

---

## Описание данных

В работе используется набор данных сетевого трафика, где каждый объект представляет собой **сетевой поток**, описанный набором числовых и категориальных признаков.

Примеры используемых характеристик:
- тип протокола (TCP, UDP и др.)
- длительность сетевого потока
- количество и размер пакетов
- скорость передачи данных
- межпакетные интервалы
- целевая метка (нормальный трафик / тип атаки)

Рассматриваемые типы атак:
- DrDoS   
- R2L (Remote-to-Local)
